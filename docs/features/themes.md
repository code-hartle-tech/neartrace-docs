# Themes

> **S01E09 — "The app that matches your vibe (and your threat level)."**

<div class="tldr">
<strong>TL;DR</strong>
10+ Material 3 themes. All dark-first. Semantic colour roles throughout — no hardcoded colours. Switch instantly from Settings.
</div>

NearTrace ships with over 10 themes built on Material Design 3's dynamic colour system. All themes use semantic roles (`colorPrimary`, `colorSurface`, etc.) so every component adapts correctly.

**Switch themes:** Settings → Themes

Some themes in the lineup:

| Theme | Vibe |
|---|---|
| **Midnight** | Pure black, cyan accent — default |
| **Carbon** | Dark grey, amber accent |
| **Infrared** | Dark + red accent — high contrast |
| **Forest** | Dark green tones |
| **Ultraviolet** | Deep purple |
| **Monochrome** | Pure greyscale for maximum contrast |

More themes on the roadmap, including a full design token system so the community can build custom skins.
